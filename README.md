# xrpl-cli

A nice CLI to interact with the TestNet XRP Ledger.

## Requirements

- Node.js (version 15)
- Typescript (latest version)

## Instructions

### Clone/Fork the repo

### Go to the xrpl-cli directory

```sh
cd /path/to/xrpl-cli/xrpl-cli
```

### Build

```shell
npm run build
```

### Run

```shell
npm start
```
